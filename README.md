#### What is Flutter?
Flutter is a mobile UI framework by Google introduced in 2011 to create Android and iOS applications.

#### What is Dart?
Dart is a programming language which is used in Flutter framework.   
- Features   
Its Object oriented, statically typed language.   
- How does it work on web and mobile devices?   
Its transpiled to JavaScript to run in web browsers and compiled to machine code to run on mobile devices.
